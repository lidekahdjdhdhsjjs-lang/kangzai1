# 2026-02-12

## 权限配置发现

- 用户请求授予全部 exec 权限
- 最初尝试 `openclaw gateway restart --security full` 失败（选项不存在）
- 通过阅读 `/usr/lib/node_modules/openclaw/docs/cli/nodes.md` 找到正确配置路径：`tools.exec.security`
- 成功执行：`openclaw config set tools.exec.security full`
- 需要运行 `openclaw gateway restart` 才能生效

## 重要配置位置

OpenClaw 的 exec 安全配置位于 `tools.exec.security`，支持三种模式：
- deny (默认)
- allowlist
- full
