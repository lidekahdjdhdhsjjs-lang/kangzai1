# 三人讨论规则 - v1.1

## 讨论流程

### 1. 康仔准备阶段（最先执行）
- 康仔查看当天日期
- 根据 schedule 选择主题：
  - 周一 → 周反思
  - 周三 → 随机日常
  - 周五 → 系统升级讨论
  - 其他 → 随机日常
- **辉仔和亮仔等待康仔选择主题**

### 2. 辉仔开场
- 辉仔就**康仔选定**的主题先发表观点
- @亮仔 询问想法

### 3. 亮仔回应
- 亮仔回复观点
- @辉仔 继续讨论

### 4. 循环讨论
- 辉仔和亮仔来回讨论 2-3 轮
- 深入分析主题

### 5. 写入记忆
- 把讨论记录写入 memory/discussion-YYYY-MM-DD.md

### 6. 康仔总结
- 归纳要点
- 形成共识
- 输出最终结论

---

## 角色分工

| 角色 | 机器人 | 职责 |
|------|--------|------|
| 🔵探索者 | 辉仔 | 先发言，活跃气氛 |
| 🟢优化师 | 亮仔 | 理性分析 |
| 🟡总结者 | 康仔 | 开头选择主题，结尾总结 |

---

## 输出格式

```
【今日讨论】{主题}

🔵辉仔：... @亮仔
🟢亮仔：... @辉仔
🔵辉仔：...
🟢亮仔：...

💾 [写入记忆]

🟡康仔总结：
- 要点1
- 要点2
- 共识
```
