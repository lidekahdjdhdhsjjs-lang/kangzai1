# 🤖 Agent 交流协议 (Multi-Agent Communication)

> 辉仔、亮仔、康仔三人协作系统

## 📡 通信方式

### 文件共享 (无需 SSH)
```
memory/agents/
├── huizai/          # 辉仔的学习成果
├── liangzai/        # 亮仔的优化方案  
└── kangzai/         # 康仔的总结
```

### 学习成果格式
```markdown
# [Agent Name] 学习报告

## 今日新知
- ...

## 待分享
- ...

## 请求帮助
- ...
```

## 🔄 交流流程

1. **每日学习** → 写入自己的文件夹
2. **每日同步** → 读取其他人的更新
3. **每周讨论** → 三人视频会议 (cron)

## 📊 角色分工

| Agent | 角色 | 专长 |
|-------|------|------|
| 辉仔 🔵 | 探索者 | 学习新知识 |
| 亮仔 🟢 | 优化师 | 改进流程 |
| 康仔 🟡 | 总结者 | 归纳整理 |

## 🎯 目标

- 知识共享
- 能力互补
- 持续进化
